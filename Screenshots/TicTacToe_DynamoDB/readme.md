DynamoDB TicTacToe Application Deployment on AWS EC2

This guide explains how to deploy a Flask-based TicTacToe application that stores game data in Amazon DynamoDB using an IAM Role attached to an EC2 instance.

⸻

1. Create IAM Role
	1.	Open AWS Console → IAM → Roles
	2.	Click Create Role
	3.	Select:

Trusted entity type: AWS Service
Use case: EC2

	4.	Click Next
	5.	Attach Permission Policy:

DynamoDBFullAccess

	6.	Click Next
	7.	Configure Role:

Role Name: Game-Role

	8.	Click Create Role

⸻

2. Launch EC2 Instance

Launch an EC2 instance with the following configuration.

Basic Configuration

AMI: Amazon Linux 2
Instance Type: t2.micro

Security Group

Allow inbound traffic on:

Port: 5000
Protocol: TCP
Source: Anywhere (0.0.0.0/0)

Advanced Details

Attach the IAM Role:

IAM Instance Profile: Game-Role

Metadata Configuration:

Metadata accessible: Enabled
Metadata Version: V1 and V2 (token optional)

This metadata will allow the instance to access credentials generated by the IAM Role.

Click Launch Instance

⸻

3. Connect to EC2 Instance

Connect using SSH.

sudo su
yum update -y


⸻

4. Install Development Tools and Dependencies

Install essential development tools required for compiling software.

sudo yum groupinstall -y "Development Tools"

Install required development libraries.

sudo yum install -y openssl-devel bzip2-devel libffi-devel


⸻

5. Install Python 2.7.18

Download Python source.

cd /usr/src
sudo wget https://www.python.org/ftp/python/2.7.18/Python-2.7.18.tgz

Extract the package.

sudo tar xzf Python-2.7.18.tgz

Enter the directory.

cd Python-2.7.18

Configure the build.

sudo ./configure --enable-optimizations

Compile and install Python without replacing system Python.

sudo make altinstall

Verify installation.

python2.7 -V

Expected output:

Python 2.7.18


⸻

6. Install pip for Python 2.7

Download pip installer.

wget https://bootstrap.pypa.io/pip/2.7/get-pip.py

Install pip.

python get-pip.py

If the above fails

Check Python 3.

python3 -V

Install pip for Python 3.

sudo dnf install python3-pip -y


⸻

7. Install Required Python Packages

Install Flask web framework.

pip install Flask

Install AWS SDK for Python.

pip install boto

Install configuration parser.

pip install configparser


⸻

8. Install Git

yum install git -y


⸻

9. Clone the TicTacToe Project

Navigate to EC2 user directory.

cd /home/ec2-user/

Clone the repository.

git clone https://github.com/KastroVKiran/DynamoDB_AWS_Project.git

Enter the repository.

cd DynamoDB_AWS_Project

List files.

ls


⸻

10. Configure Application

Edit configuration file.

vi config

Update:

region
endpoint

Save and exit.

ESC
:wq

Edit second configuration file.

vi config.ini

Update the region.

Save and exit.

ESC
:wq


⸻

11. Run the Application

Execute the application.

python application.py \
--config config.ini \
--mode service \
--endpoint dynamodb.ap-south-1.amazonaws.com \
--serverPort 5000


⸻

12. Access the Application

Open a browser.

http://<EC2-Public-IP>:5000

You will see the TicTacToe Application UI.

⸻

13. Create DynamoDB Table
	1.	Enter a username.
	2.	Click Login
	3.	Click Create Table

A DynamoDB table named:

Games

will be created automatically.

This happens because of the script:

setupDynamoDB.py


⸻

14. Verify DynamoDB Table

Open AWS Console → DynamoDB

Open table:

Games

Initially:

No Items


⸻

15. Play Multiplayer Game

Open the application in two browsers.

Example:

Chrome
Firefox

Login with different usernames.

Steps
	1.	Player 1 → Click Create
	2.	Enter Player 2 username
	3.	Player 2 → Refresh page
	4.	Accept game request
	5.	Start playing

⸻

16. Verify Game Data in DynamoDB

Go to:

AWS Console → DynamoDB → Tables → Games

Click:

Scan or Query Items

Select:

Scan

Click:

Run

You will see:

Game details
Players
Moves
Game state

Click an item to see all game moves stored in DynamoDB.
